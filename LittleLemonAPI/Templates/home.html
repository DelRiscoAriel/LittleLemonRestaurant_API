{% load static %} Little Lemon Restaurant API

Description

This ia an API project for a restaurant. People with different roles will be able to browse, add and edit menu items, place orders, browse orders, assign delivery crew to orders and finally deliver the orders. Throttling, filtering, pagination and sorting capabilities are implemented.

Links

Function

Sample Input

admin/

Adim Interface to manage the API.

auth/users/

A super user or manager can List all user accounts, while others can only List their account. Also to reguister new users.

auth/users/me/

List the current user account. Also to Update or Delete account.

auth/token/login/

Generates access tokens that can be used in other API calls providing the correct username and password

api/categories

A super user or manager user can List and Create Categories, while other can only List Categories

api/categories/{Category ID Number}

A super user or manager user can List, Update or Delete the Category, while other can only List the Category

api/menu-items

A super user or manager user can List and Create Menu Items, while other can only List Menu Items

api/menu-items/{Menu Item ID Number}

A super user or manager user can List, Update or Delete the Menu Item, while other can only List Menu Item

api/groups/manager/users

Only super users and managers can List, Add or Remove users from the Manager Group

{"username" : "User Name"}

api/groups/delivery-crew/users

Only super users and managers can List, Add or Remove users from the Delivery Group

{"username" : "User Name"}

cart/menu-items

User can List, Add or Remove Menu Items from their Cart

{"menuitem_id" : Menu Item ID,
"quantity" : Number of Items}

orders

Users can List their Orders and Create an Order from their Cart with an empty POST request

Deliveries can all List the Orders assigned to them

Managers can List all Orders

To paginate add ?perpage=DesiredItemsPerPage&page=PageNumber

orders/{Order ID Number}

Users can List the order if it is theirs

Deliveris can List the Order if it is assigned to them and Update the status of the Order

Managers can List the Order, Assign a Delivery to the order and Delete the Order

For Deliveries
{"status" : False or Trues}

For Managers
{"delivery_id" : Delivery User ID}
